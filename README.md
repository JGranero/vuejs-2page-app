# Project setup
```
Vue CLI: @vue/cli 4.5.11
Node: n v12.6.0
Yarn: yarn install
Json-Server: npm install -g json-server
             curl -o db-initial.json https://tracktik-challenge.staffr.com/db
             cp db-initial.json db.json
             json-server --watch db.json
```
